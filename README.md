# 🦅 Geryon Browser

**Приватный, лёгкий и настраиваемый браузер на движке Chromium (QtWebEngine).**

Полноценный, самостоятельный браузер, сохранивший главную идею — **никакой телеметрии, никаких внешних сервисов, только локальные данные**.

## ✨ Почему Geryon?
- **Полная приватность** — без Google Safe Browsing, Crash Reports и прочего шпионажа.
- **Блокировщик рекламы + XSS-фильтр** на уровне запросов.
- **Менеджер паролей** с мастер-паролем и шифрованием AES-256.
- **Защита от цифрового отпечатка** (User-Agent, WebGL, Canvas).
- **Трекер времени** — отчёт, на каких сайтах вы сидите дольше всего.
- **Режим для слабовидящих** (крупный шрифт, высокий контраст).
- **QR-код страницы**, Инкогнито, Калькулятор в адресной строке.
- **Стартовая панель** с погодой и курсами валют.
- **Портативный AppImage** — не требует установки, все зависимости внутри.


## 📥 Установка
1. Скачайте AppImage.
2. Сделайте исполняемым: `chmod +x Geryon-x86_64.AppImage`
3. Запустите: `./Geryon-x86_64.AppImage`

## 🔧 Для разработчиков
Если хотите собрать из исходников, необходимые зависимости перечислены в прилагаемом файле `requirements.txt`.
